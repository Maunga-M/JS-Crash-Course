What you learned

Every concept built on the last.

Here is everything that you covered.

01
Introduction to JavaScript

You learned what JavaScript is and how it runs in the browser. You declared variables using let, worked with the primitive data types, and used typeof to inspect values. You also covered naming conventions and why descriptive names matter.

02
Arithmetic

You worked through the core arithmetic operators and learned shorthand assignment syntax. You explored how operator precedence determines the order in which expressions are evaluated.

03
Strings

You explored how strings work in JavaScript including quoting rules, concatenation, and template strings. You learned to index into a string and use its length property. You also learned to apply a set of built-in string methods for transforming and searching text.

04
Type Conversion

You learned the difference between implicit coercion and explicit conversion. You studied truthy and falsy values and how they affect conditional logic, and used dedicated functions to convert between data types.

05
Comparison and Logical Operators

You worked through all the comparison operators and the important distinction between loose and strict (in)equality. You learned the three logical operators and explored short-circuit evaluation and operator precedence.

06
Conditionals

You learned to control program flow with if, else if, and else. You understood how JavaScript evaluates conditions top to bottom and stops at the first match. You also the ternary operator as an alternative to writing a short if/else statement.

07
Arrays

You learned to store multiple values in a single variable. You accessed and updated elements by index, mutated arrays using built-in methods, and understood the difference between copying by reference and copying by value. You also covered how to search an array to get a specific element or its index.

08
Loops

You learned three loop structures. The for loop for when the iteration count is known. The while loop for when it depends on a condition. And for...of for clean iteration over arrays. You combined loops with arrays to compute totals and filter results.

09
Functions

You learned to define reusable blocks of code, call them by name, and pass values in through parameters and arguments. You used the return keyword to send values back out and were introduced to the concept of scope.

10
Objects

You learned to group related data as named key-value pairs using object literals. You accessed properties with dot and bracket notation. You learned how to add, update and delete properties. You checked for key existence using the in operator and looped over objects. You also got a glimpse of how arrays and objects can be used together.

Where to go from here

Now that you have the foundations, here are some of the best free resources you can use to become a real javascript developer.

Reference
MDN Web Docs
The definitive reference for JavaScript. Every method, every syntax, every edge case. Bookmark this -- you will use it constantly throughout your career.
developer.mozilla.org
Deep Dive
The Modern JavaScript Tutorial
One of the most thorough free JavaScript guides available. Goes deep on every topic from this course and far beyond, with clear explanations and exercises.
javascript.info
Curriculum
freeCodeCamp JavaScript
A structured, project-based curriculum that takes you from foundations to algorithms and data structures. Completely free, browser-based, and earns a certificate.
freecodecamp.org
Interactive
Scrimba JavaScript
Video lessons where you can pause and edit the instructor's code directly in the browser. A great next step if you learn well by doing alongside someone.
scrimba.com
Full Stack
The Odin Project
A full-stack curriculum that takes JavaScript seriously. Project-heavy, community-backed, and used by thousands of people who have gone on to work as professional developers.
theodinproject.com
